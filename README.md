# LineStartMinimized
- Startt Minimized.
- Remove splash screen.

# How to use
### Install
```
> LineStartMinimized.exe install linelauncher.exe
Succeeded
```

### Uninstall
```
> LineStartMinimized.exe uninstall linelauncher.exe
Succeeded
```

# How to build
1. Install Detours
```
vcpkg install detours:x86-windows-static
```

2. Build with Visual Studio
